Smart Calculator

Tap on the microphone and then start Speaking.
Voice commands can be spoken in any order:
(+) - "Plus"
(-) - "Minus"
(/) - "Divided by"
(x) - "Multiply"
(^) - "To the power of"
Eg: 2 multiply 5 will be 2 × 5
This app is based on an experimental API
